# MoView

## Overview
ini adalah aplikasi copycat dari letterboxd yang berfungsi untuk melihat sinopsis dan rating film secara realtime
yang terintegrasi menggunakan API dari TMDB

## Fitur
1. Informasi film trending secara real time
2. sinopsis dan informasi singkat terkait film dan genre
3. klasifikasi berdasarkan kategori film/series
4. search bar yang berfungsi mencari judul film agar menampilkan informasi 

## API
API yang digunakan berasal dari website TMDB
